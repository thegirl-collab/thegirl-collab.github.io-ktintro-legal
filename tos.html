2. Bot Functionality

KTIntro provides a customizable introduction template system for Discord servers. Admins configure questions; members fill them out via modal, and the Bot posts formatted embeds in a dedicated channel.

3. Configuration & Permissions

  • The Bot may request Manage Channels to auto-create an #introductions channel. Without it, admins must create that channel and grant Send Messages & Embed Links.
  • All configuration is per-guild; no cross-server data sharing occurs.

4. Privacy & Data Storage

The Bot stores only the following per-guild data in a local SQLite database:

  • Guild ID
  • Channel ID
  • Configured question templates

No personal user data is retained beyond what Discord provides at runtime for slash-command interactions. You may delete your server’s data by removing the Bot.
